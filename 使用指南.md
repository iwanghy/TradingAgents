## 环境准备

### 步骤 1：克隆项目并创建环境

```bash
# 1. 克隆项目（如果还没有）
git clone https://github.com/TauricResearch/TradingAgents.git
cd TradingAgents

# 2. 创建并激活 conda 环境
conda create -n tradingagents python=3.10 -y
conda activate tradingagents

# 3. 安装依赖
pip install -r requirements.txt
```

**⚠️ 重要提示：每次运行前都要激活环境**
```bash
conda activate tradingagents
```

## API 密钥配置

### 步骤 2：配置 LLM API 密钥

**至少需要配置一个 LLM 提供商的 API 密钥**

#### 选项 A：使用 .env 文件（推荐）

```bash
# 复制示例文件
cp .env.example .env

# 编辑 .env 文件
nano .env  # 或使用你喜欢的编辑器
```

在 `.env` 文件中添加你的 API 密钥：

```bash
# 选择一个或多个配置：

# OpenAI (GPT) - 最强大
OPENAI_API_KEY=sk-your-openai-key-here

# Google (Gemini) - 性价比高
GOOGLE_API_KEY=your-google-api-key-here

# Anthropic (Claude)
ANTHROPIC_API_KEY=sk-ant-your-key-here

# xAI (Grok)
XAI_API_KEY=your-xai-key-here

# OpenRouter (多模型聚合)
OPENROUTER_API_KEY=your-openrouter-key-here

# 数据源（可选，默认使用免费的 yfinance）
ALPHA_VANTAGE_API_KEY=your-alpha-vantage-key-here
```

#### 选项 B：使用环境变量

```bash
export OPENAI_API_KEY=sk-your-key-here
export GOOGLE_API_KEY=your-key-here
# ... 其他密钥
```

## 方法一：使用 CLI 界面（推荐新手）

### 步骤 3：启动 CLI

```bash
# 确保环境已激活
conda activate tradingagents

# 启动 CLI
python -m cli.main
```

### 步骤 4：在 CLI 中选择配置

CLI 会显示交互式界面，让你选择：

#### 4.1 选择股票代码
- 输入股票代码，例如：`AAPL`、`NVDA`、`TSLA`
- 支持美股、港股等

#### 4.2 选择分析日期
- 输入格式：`YYYY-MM-DD`
- 例如：`2024-01-15`
- 建议选择最近的交易日

#### 4.3 选择分析师团队
可以选择以下分析师（可多选）：
- **Market Analyst** - 技术分析（MACD、RSI 等）
- **Social Analyst** - 社交媒体情绪分析
- **News Analyst** - 新闻分析
- **Fundamentals Analyst** - 基本面分析

**建议**：首次使用全选，获得全面分析

#### 4.4 选择 LLM 提供商
根据你配置的 API 密钥选择：
- `openai` - OpenAI GPT
- `google` - Google Gemini
- `anthropic` - Anthropic Claude
- `xai` - xAI Grok
- `openrouter` - OpenRouter
- `ollama` - 本地模型

#### 4.5 选择模型
- **Deep Think LLM** - 用于复杂推理的主模型
  - OpenAI: `gpt-5.2`, `gpt-5-mini`
  - Google: `gemini-2.5-pro`, `gemini-2.5-flash`
  - Anthropic: `claude-4-opus`, `claude-4-sonnet`
  - xAI: `grok-4`

- **Quick Think LLM** - 用于快速任务的轻量模型
  - 建议选择更快的模型以节省成本

#### 4.6 设置研究深度
- **Debate Rounds** - 辩论轮数（默认 1）
  - `1` - 快速分析
  - `2-3` - 深入分析（推荐）
  - `4+` - 最深入分析（耗时更长）

### 步骤 5：查看实时分析过程

CLI 会显示：
```
┌─ TradingAgents Analysis ─────────────┐
│                                       │
│  📊 Market Analyst        [Running]  │
│  💬 Social Analyst       [Pending]  │
```

每个分析师会：

### 步骤 6：查看最终结果

分析完成后，你会看到：

**投资建议**：
- ✅ **BUY** - 建议买入
- ⏸️ **HOLD** - 建议持有
- ❌ **SELL** - 建议卖出

**详细报告**：
- 各分析师的观点
- 研究员团队的多空辩论
- 风险评估
- 最终投资决策

## 方法二：编写 Python 脚本（推荐进阶用户）

### 步骤 3：创建分析脚本

创建文件 `analyze_stock.py`：

```python
from tradingagents.graph.trading_graph import TradingAgentsGraph
from tradingagents.default_config import DEFAULT_CONFIG
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

# 步骤 1: 配置参数
config = DEFAULT_CONFIG.copy()

# 配置 LLM 提供商
config["llm_provider"] = "openai"  # 或 google, anthropic, xai, openrouter
config["deep_think_llm"] = "gpt-5-mini"     # 主模型
config["quick_think_llm"] = "gpt-5-mini"  # 快速模型

# 配置分析深度
config["max_debate_rounds"] = 2  # 辩论轮数，1-4

# 配置数据源（可选，默认 yfinance）
config["data_vendors"] = {
    "core_stock_apis": "yfinance",
    "technical_indicators": "yfinance",
    "fundamental_data": "yfinance",
    "news_data": "yfinance",
}

# 步骤 2: 选择分析师（可选）
# 默认包含所有分析师，你可以自定义
# 可选: "market", "social", "news", "fundamentals"

# 步骤 3: 初始化分析系统
ta = TradingAgentsGraph(debug=True, config=config)

# 步骤 4: 运行分析
ticker = "NVDA"          # 股票代码
trade_date = "2024-01-15"  # 分析日期

print(f"正在分析 {ticker} ({trade_date})...\n")

# 执行前向传播
state, decision = ta.propagate(ticker, trade_date)

# 步骤 5: 查看结果
print("\n" + "="*50)
print("最终决策:", decision)
print("="*50 + "\n")

# 可选：查看完整状态
# print("完整状态:", state)
```

### 步骤 4：运行脚本

```bash
# 确保环境已激活
conda activate tradingagents

# 运行分析脚本
python analyze_stock.py
```

## 常见问题

### Q1: 分析需要多长时间？

**答**：取决于配置
- **快速分析**（1 轮辩论）：2-5 分钟
- **标准分析**（2 轮辩论）：5-10 分钟
- **深度分析**（3-4 轮辩论）：10-20 分钟

### Q2: 需要多少 API 费用？

**答**：取决于提供商和配置
- **OpenAI GPT-5-mini**：约 $0.01-0.05/次分析
- **Google Gemini Flash**：约 $0.005-0.02/次分析
- **深度分析（更多辩论）**：费用相应增加

### Q3: 分析准确吗？

**答**：重要提示
- ⚠️ 本框架用于**研究目的**
- ⚠️ **不构成投资建议**
- 分析质量取决于：
  - LLM 模型能力
  - 数据质量
  - 分析深度设置
  - 市场复杂度

### Q4: 可以分析哪些股票？

**答**：支持大多数公开交易股票
- ✅ 美股（NYSE, NASDAQ）
- ✅ 港股
- ✅ A 股（部分）
- ❌ 加密货币（需要特殊配置）
- ❌ 小盘股（数据可能不足）

### Q5: 如何获得更好的结果？

**答**：优化建议
1. **增加辩论轮数**：设置 `max_debate_rounds=2` 或 `3`
2. **使用更强的模型**：选择 `gpt-5.2` 而非 `gpt-5-mini`
3. **选择所有分析师**：获得多角度分析
4. **使用高质量数据源**：配置 Alpha Vantage（需要 API 密钥）
5. **选择合适的日期**：避免节假日或周末

### Q6: API 密钥在哪里获取？

**答**：
- **OpenAI**: https://platform.openai.com/api-keys
- **Google**: https://console.cloud.google.com/apis/credentials
- **Anthropic**: https://console.anthropic.com/settings/keys
- **xAI**: https://console.x.ai/
- **Alpha Vantage**: https://www.alphavantage.co/support/#api-key

### Q7: 遇到错误怎么办？

**答**：常见问题排查

**错误 1: API key not found**
```bash
# 检查 .env 文件是否存在
ls -la .env

# 检查 .env 文件内容
cat .env
```

**错误 2: Module not found**
```bash
# 确保在 conda 环境中
conda activate tradingagents

# 重新安装依赖
pip install -r requirements.txt
```

**错误 3: Rate limit exceeded**
- 等待一段时间后重试
- 或切换到不同的 LLM 提供商
