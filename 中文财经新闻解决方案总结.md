## 📋 AKShare 新闻功能详解

### 1. 东方财富个股新闻

**函数**: `ak.stock_news_em(symbol="600519")`

**数据字段**:
- 新闻标题
- 新闻内容
- 发布时间
- 文章来源
- 新闻链接
- 关键词

**示例**:
```python
import akshare as ak

# 获取贵州茅台新闻
news = ak.stock_news_em(symbol="600519")
print(f"获取到 {len(news)} 条新闻")

# 显示最新3条
for idx, row in news.head(3).iterrows():
    print(f"标题: {row['新闻标题']}")
    print(f"时间: {row['发布时间']}")
    print(f"来源: {row['文章来源']}\n")
```

### 2. 财新网新闻

**函数**: `ak.stock_news_main_cx()`

**特点**:
- 财新网是高质量财经媒体
- 深度分析和专业报道
- 适合长期投资分析

### 3. CCTV新闻联播

**函数**: `ak.news_cctv(date="20240424")`

**特点**:
- 官方新闻媒体
- 宏观经济政策
- 权威性高

### 4. 百度经济日历

**函数**: `ak.news_economic_baidu(date="20251126")`

**特点**:
- 宏观经济数据
- 重要经济事件
- 预期vs实际对比
